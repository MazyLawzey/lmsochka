# 📚 ЛМСОЧКА - Руководство по фронтенду

## 🎯 Обзор фронтенда

Профессиональный веб-интерфейс для LMS, разработанный с использованием:
- **Vue.js 3** - реактивный фреймворк
- **Axios** - HTTP клиент для API запросов
- **Bootstrap 5** - UI компоненты
- **Modern CSS** - адаптивный дизайн

## 📖 Структура фронтенда

### Главные компоненты

```
index.html
├── Auth View (Аутентификация)
│   ├── Login Form (Форма входа)
│   └── Register Form (Форма регистрации)
├── Dashboard View (Панель управления)
│   ├── Student Dashboard (Панель студента)
│   │   ├── Courses Tab (Мои курсы)
│   │   ├── Assignments Tab (Мои задания)
│   │   └── Grades Tab (Оценки)
│   ├── Teacher Dashboard (Панель преподавателя)
```

## 🔐 Аутентификация

### Вход в систему

1. Откройте `http://localhost:8000` в браузере
2. Введите имя пользователя и пароль
3. Нажмите "Войти"
4. Токен будет сохранён в `localStorage`

### Регистрация

1. Нажмите "Создать учётную запись"
2. Заполните все поля:
   - Имя пользователя (уникальное)
   - Email
   - Полное имя
   - Пароль
   - Роль (Студент/Преподаватель)
3. Нажмите "Зарегистрироваться"

### Сохранение сессии

После входа токен автоматически:
- Сохраняется в `localStorage` под ключом `accessToken`
- При перезагрузке страницы автоматически восстанавливается
- Используется для всех запросов к API

## 📡 API Интеграция

### Авторизованные запросы

Все запросы, кроме `/auth/login` и `/auth/register`, требуют токена:

```javascript
headers: {
    'Authorization': `Bearer ${accessToken}`
}
```

### Главные API эндпоинты

```
POST   /api/auth/register        - Регистрация
POST   /api/auth/login           - Вход
GET    /api/auth/me              - Информация о текущем пользователе
GET    /api/users/my-courses     - Мои курсы (для студентов)
```

## 🐛 Отладка

### Проверка сохранённого токена

Откройте консоль браузера и выполните:
```javascript
localStorage.getItem('accessToken')
```

### Проверка сетевых запросов

1. Откройте DevTools (F12)
2. Перейдите на вкладку "Network"
3. Выполните действие в приложении
4. Проверьте статус ответа (должен быть 200)

### Частые ошибки

**CORS Error:**
- Убедитесь, что на бэкенде правильно настроены CORS
- Проверьте, что API_URL в фронтенде верный

**401 Unauthorized:**
- Токен истёк - требуется новый вход
- Токен неверный - переоткройте страницу

**Network Error:**
- Сервер не запущен
- Неверный URL API

## 🔧 Расширение функционала

Для добавления новых компонентов:

1. **Добавить HTML разметку** в нужную секцию dashboard
2. **Добавить Vue данные** в `setup()` функцию
3. **Добавить методы** для обработки событий
4. **Добавить CSS стили** в `<style>` блок
5. **Добавить API запросы** через axios

Пример:

```javascript
// Добавить в data
const newData = ref([]);

// Добавить метод
async function fetchNewData() {
    const response = await axios.get(`${API_URL}/new-endpoint`, { headers });
    newData.value = response.data;
}

// Вызвать при загрузке
onMounted(() => {
    fetchNewData();
});
```
